# 모바일 청첩장

React와 Tailwind CSS로 제작한 모바일 전용 청첩장 프로젝트입니다. 숲과 초록 들판의 웨딩 커버 이미지를 중심으로, 실제 모바일 청첩장에서 자주 사용하는 초대글, 일정, 갤러리, 오시는 길, 계좌 정보, 참석 의사 전달 기능을 구성했습니다.

## 프로젝트 주제

김준호와 이수연의 결혼식을 소개하는 모바일 청첩장입니다. 단순히 정보를 나열하는 화면이 아니라, 스크롤할 때 섹션이 부드럽게 등장하는 랜딩페이지형 청첩장을 목표로 제작했습니다.

## 구현된 기능

- 모바일 화면 기준의 세로형 청첩장 레이아웃
- 메인 커버 이미지 표시 및 예식 날짜/장소 안내
- 스크롤 시 섹션이 부드럽게 등장하는 애니메이션
- 초대글과 혼주 정보 표시
- 2026년 10월 31일 예식 날짜 달력 표시
- 웨딩 분위기의 갤러리 이미지 표시
- 예식장 주소와 Google 지도 표시
- 네이버 지도 열기 기능
- 예식장 주소 복사 기능
- 신랑측/신부측 계좌 정보 펼침 애니메이션
- 계좌 정보 클립보드 복사 기능
- 참석 의사 체크 모달
- 참석자 구분, 참석 여부, 이름, 휴대폰 번호 뒤 4자리 입력
- 참석 응답 localStorage 저장 및 최근 응답 표시
- 청첩장 링크 공유 기능
- 예식 일정 메모 파일 다운로드 기능

## 사용 기술

- React
- Vite
- Tailwind CSS
- localStorage
- Web Share API
- Google Maps iframe

## 실행 방법

```bash
npm install
npm.cmd run dev
```

PowerShell에서 `npm` 실행이 막히는 경우가 있어 Windows 환경에서는 `npm.cmd` 명령어 사용을 권장합니다.
